Latest News & Updates

Welcome to LAMS 2.2!

This new release has lots of new features and improvements including Text-based Branching, so teachers can create branching conditions based on text output from tools.
LAMS 2.2 includes improvements like Learner's selection for groups, Opening gates based on conditions so learners can select which group they want to take part of, teachers can add learning outcomes to their sequences using the new Learning Outcomes/Competencies editor, get informed when certain events occur with Email Notifications, find out who is doing a lesson at the same time with Presence, use external LMS/VLE tools within LAMS via the External tool wrappers, and a lot more.
There are four new tools in LAMS 2.2: a wiki to create collaborative documents, a data collection tool, a Google Maps tool, and an online spreadsheet tool. There is also optional integration with Dimdim for whiteboard, desktop sharing, audio-video conferencing (it might require a fee).
We recommend you to join the LAMS Community where teachers and developers share their experiences of using LAMS, including a large repository of LAMS sequences shared from all around the world - a great place to start for good teaching ideas!
You can provide feedback on LAMS here, and view the LAMS Roadmap for upcoming features.
