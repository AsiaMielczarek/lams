Latest News & Updates

Welcome to LAMS 2.1!

This new release is packed with new features and improvements including Branching, so teachers can create powerful learning designs to support many different educational approaches.
LAMS 2.1 can use video and audio in every single activity to make the learner's experience richer and more interactive. LAMS has new support for Mobile Devices so learning can be taken anywhere.
Have a look at the full list of features, and if you are new to LAMS, see the LAMS Documentation for quick tutorials, animations and full details.
We recommend you to join the LAMS Community where teachers and developers share their experiences of using LAMS, including a large repository of LAMS sequences shared from all around the world - a great place to start for good teaching ideas!
You can provide feedback on LAMS here, and view the LAMS Roadmap for upcoming features.
LAMS is translated to 25 languages by more than 50 volunteers. Want to help out?
