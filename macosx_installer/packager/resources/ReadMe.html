Installing LAMS 2.3.4

IMPORTANT:

Once installed, you will be able to login into LAMS using:

Username:     test1

Password:      test1

As soon as you login, you will be able to see some lessons on how to use LAMS. There's also another three demo users: test2, test3 and test4. The passwords are the same as the usernames. 

The system administrator account is sysadmin, password: sysadmin. Use this user to change configuration settings, language, etc. 

Once the installation is complete, You will see a LAMS application in your Application's folder where you can start and stop LAMS.

If you have any problems or comments, see the LAMS Community.
